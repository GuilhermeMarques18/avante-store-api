# avante-store-api
The Avante Store API is a backend application developed in Java with Spring Boot, offering a complete REST API for product and category management.
This project was created as part of the Challenge for the Projects Sector at Avante Tech JR 🚀, focusing on applying backend development best practices, layered architecture, and integration with a relational database MySQL.

The API supports all CRUD (Create, Read, Update, Delete) operations for both products and categories, featuring:

Robust validations

Comprehensive error handling

Automatic API documentation via Swagger/OpenAPI

This project demonstrates proficiency in building scalable and maintainable backend systems.
